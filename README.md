# Jiu Timer Pro 🥋⏱️

Aplicativo mobile moderno para controle de treinos de Jiu-Jitsu, focado em atletas, professores e academias.

## 🚀 Funcionalidades

- **Cronômetro de Rola:** Configure tempo de rola, descanso, preparação e quantidade de rounds.
- **Treino de Gás:** Protocolos específicos como Shark Tank, Gás para Campeonato e HIIT.
- **Configurações Salvas:** Salve seus treinos favoritos para acesso rápido.
- **Histórico de Treinos:** Acompanhe seu volume de treino e evolução.
- **Interface Esportiva:** Design de alto contraste (Dark Mode) otimizado para uso durante o treino.
- **Alertas:** Feedback sonoro e tátil (vibração) nas transições de fase.

## 🛠️ Tecnologias

- React Native (Expo)
- TypeScript
- React Navigation
- AsyncStorage (Persistência local)
- Lucide React Native (Ícones)
- Expo Haptics & AV (Feedback sensorial)

## 📦 Instalação

1. Clone o repositório:
```bash
gh repo clone emanoeltga/cronometro
```

2. Instale as dependências:
```bash
pnpm install
```

3. Inicie o projeto:
```bash
pnpm start
```

## 📱 Telas Principais

- **Dashboard:** Resumo do dia e acesso rápido.
- **Configuração:** Personalização total do treino.
- **Execução:** Cronômetro em tela cheia com cores indicativas de fase.
- **Histórico:** Estatísticas semanais e diárias.
